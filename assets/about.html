FantastischMemo Free Edition

Ver 1.5.3
Author: Haowen Ning, Qi Wang

1. Introduction

FantastischMemo is a spaced repetition flashcard learning software. It implements a slightly modified Mnemosyne algorithm to maximize the learning efficient. The key features are as follows:

  • Well tested and highly efficient Mnemosyne algorithm for fast learning
  • Text-to-Speech feature to aid language learning. Support multiple languages
  • Convert t he Mnemosyne XML directly to database on the device.
  • Edit or create the database on device
  • PC tool is provided to convert Mnemosyne XML, CSV, tab TXT file to the database
  • A collection of database is provided in Market to try out.
  • Text format can be changed to fit different database and screen size.
  • Each flashcard can be edited or deleted on the device.
  • Simple, easy-to-use and power saving interface.

3. How to use

3.1 Install/Open/Close Databases

To install a database, you can use the Use the FantastischMemo DB Installer in the market to get a collection of converted database. It will automatically copy db to your sdcard/fantastischmemo directory. Or you can convert the Mnemosyne XML or CSV or TXT (Tab separation) files to anywhere in SD card.

The program also have an XML importing function built in. You can convert Mnemosyne's XML file to the database.

To open a database, you just press Open Database and select the .db file in the ugly file browser or one from recent list.

To close the program, you can just press the back key or select back from the menu and "Exit" from the main screen.

3.2 How to memorize the flashcard

In the flashcard screen, you will see the question with the answer hidden. When you press the screen, the answer will be shown and you need to grade yourself by clicking the grading button. The meanings of the grades are:

  • Grade 0: You have forgotten the card completely.
  • Grade 1: You have forgotten the card completely.
  • Grade 2: Barely correct answer.
  • Grade 3: Correct answer with much effort.
  • Grade 4: Correct answer with some effort
  • Grade 5: Correct answer without any difficulties.

After selecting the grade, the next question will be shown. The title bar will show the number of scheduled cards for repetition and the new cards left. You need to decide how long to use this software each day. Generally speaking 30-60min is the best.

3.3 How to use the Text-to-Speech function

Text-to-Speech function is available for Android OS 1.6+ and it currently can speak English (US, UK), Italian, German, French and Spanish. To use this function, you can set up the automatic speech function in the Option in main screen so that every question answer will be spoken out according to the Locale information in the database. If you do not want automatic speaking, you can also let the phone read the individual word in the menu of the flashcard screen.

3.4 How to edit the database

In the main screen you can use "Create/Edit Database" to edit databases. In the file browser screen you can create the new database through the menu.

This program also provides some basic function to edit the database through the "Settings" and "Detail" menu items in flashcard screen. In "Settings" screen you can edit the format of the question and answer, and also the locale of them. In "Detail" screen, you can edit anything or delete the item.
